Introduction

The primary goal of this project is to provide an open-access resource for learning single-variable calculus according to the following criteria:

  1. Present calculus using the infinitesimal method
  2. Include interactive graphics for active learning
  3. Ensure materials are easily and freely accessible

To achieve these goals, the final product is this website: a public collection of reading and practice materials for learning and applying single-variable calculus.

For the reader, this website can be a standalone resource for independent study or a reference text for a particular topic. In the first case, read the materials in the order they appear; in the second case, visit the relevant section(s) for your topic — each section will reference any required knowledge from previous sections.
